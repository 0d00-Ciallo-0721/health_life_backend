# 🤖 AI Service Domain (人工智能服务)

> **目录位置**: `/apps/aiservice/`
> **模块定位**: 对接外部大模型 API，为应用提供视觉理解和逻辑推理能力。

## 📖 1. 核心功能

* **视觉识别**: 调用 `Qwen3-VL` 模型，解析食物图片，输出结构化 JSON (菜名、热量、宏量营养素)。
* **逻辑推理**: 基于用户的生理档案 (Profile) 和 饮食日志 (Intake)，生成自然语言建议。

## 🔧 2. 技术实现

* **服务隔离**: 所有 AI 逻辑封装在 `AIService` 类中，与业务 View 解耦。
* **Prompt Engineering**:
    * **识图**: 强制模型输出纯 JSON 格式，便于后端解析。
    * **建议**: 注入 "语气亲切"、"鼓励为主" 的 System Prompt。
* **容错机制**: 当大模型 API 超时或报错时，返回友好的兜底提示，不影响主流程。
